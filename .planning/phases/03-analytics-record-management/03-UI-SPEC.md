## Interaction Patterns

### Date Range Picker (AN-05)
- Use `el-date-picker` with `type="daterange"`, `value-format="YYYY-MM-DD"`
- Default range: last 30 days
- On change: reload all charts and stats with new date bounds
- Pass `startDate` and `endDate` as query params to analytics API

### Period Toggle (7天 / 30天)
- Two pill buttons: "7天" and "30天"
- Default: 7天
- On toggle: switch chart data window, reload chart
- Do NOT show custom period in toggle — custom is via date range picker

### Chart Resize on Tab Switch
- When switching el-tabs to a new metric category, call `chartInstance.resize()` inside `nextTick()`
- Pattern: Home.vue lines 500-525 shows established ECharts init + setOption pattern

### Delete Record Flow (DM-03, already implemented)
- User clicks delete button in el-table
- `ElMessageBox.confirm('确认删除该记录吗?', '提示', { type: 'warning' })`
- On confirm: `request.delete(\`/record/${type}/${id}\`).then(...)`
- On success: `ElMessage.success('删除成功')` + reload data
- On error: `ElMessage.error(res.msg)`

### Edit Record Flow (DM-02, already implemented)
- User clicks edit button in el-table
- `handleEdit()` copies row data into `form` reactive object
- Opens dialog with pre-populated fields
- Save calls `request.post(\`/record/${type}/save\`, form)`
